🛠️ Preparando el entorno Docker

Una de las primeras tareas al recibir tu Mini PC es configurar el entorno de contenedores. Aquí tienes el comando rápido para actualizar y preparar tu sistema en Debian/Ubuntu:

bash — root@arkeon-lab
$ sudo apt update && sudo apt upgrade -y
$ curl -fsSL https://get.docker.com -o get-docker.sh
$ sudo sh get-docker.sh
$ sudo usermod -aG docker $USER
